# Publish to GitHub

Target repository:

https://github.com/ht1505/Road-Accident-Analysis

## Steps

1. Clone your repository locally or initialize git in this folder.
2. Copy this structured project content into the repository root.
3. Commit and push.

## Suggested Commands

git init
git add .
git commit -m "Initial professional project structure: Python, KNIME, Tableau assets"
git branch -M main
git remote add origin https://github.com/ht1505/Road-Accident-Analysis.git
git push -u origin main

## Post-push verification checklist

- README renders all dashboard and worksheet preview images.
- KNIME workflow SVG opens correctly in browser.
- data/raw/Road.csv is present.
- notebooks/Road_Accident.ipynb is present.
- src/run_pipeline.py runs successfully.
