# Project Overview

This project analyzes Indian road accident records to understand severity patterns and risk factors.

## Core Objectives

- Clean and preprocess accident records for reliable analysis.
- Explore severity distribution by time, location, and driver/environment features.
- Build visual analytics in Tableau for stakeholder-friendly reporting.
- Build and evaluate baseline classification models in KNIME.

## Tools Used

- Python: data cleaning, transformation, export automation
- Pandas and NumPy: structured data handling
- KNIME: no-code/low-code workflow for EDA and ML
- Tableau: worksheet-level and dashboard-level visual storytelling
